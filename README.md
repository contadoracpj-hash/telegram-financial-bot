# telegram-financial-bot
This is a financial telegram bot where i can track my daily expenses, like supermarket, car, etc. It's connected to a postgres database, using node.js and typescript with telegraf.

I've made this bot to know how much money i spend in a day, a week, or a month. and what categories i spend money on. It is only for daily expenses, not for income or savings or investments and not for fix bills like rent, subscriptions, internet, etc.
